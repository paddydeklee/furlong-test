Login specification

Tags: login, access, security

The user must be able to log in securely using username and password

  • User must be logged out
  • User must have correct credentials

Unsuccessful login

Tags: unsuccessful

If the user enters the incorrect details, the user will remain logged out

  • Login as user <username> and password <password>
  • The user will be logged out

Successful login

Tags: successful

If the user enters the correct details, the user will be logged in (the user is still on the login page at this point)

  • Enter correct credentials
  • The user will be logged in
